配置文件名为xxxfile

有很多技术、工具和框架，去给自己的配置文件命名的时候，用：xxxfile[.yyy]

比如：

前端开发

  • grunt：Gruntfile.js
  • gulp：guplfile.js

iOS开发

  • Carthage：Cartfile
  • Cocoapods：Podfile
